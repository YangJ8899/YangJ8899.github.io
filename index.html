ShoppingQ

ShoppingQ was a web application for the Def Hacks Virtual 2020 hackathon. The main premise was to create a user friendly web app that could be useful for the covid-19 pandemic. Me and my classmates decided to create an application that allows stores to enter how many people are waiting in line, and will calculate the average wait time to enter the store. My role was to create a security system for the store owners with SQL and PHP. Things like signing up, logging in, and resetting a password. I also worked on creating a secure database to store all the information that would be inputted into the application.
